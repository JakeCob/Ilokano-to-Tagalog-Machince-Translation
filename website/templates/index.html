About

A system developed by a group of computer science students from PUP Manila using some of the following languages and tools:

  • Python
  • Jupyter Notebook
  • HTML, CSS, Javascript

Ilokano and Tagalog are local languages in the Philippines and are two of the most spoken languages in the country. The system is a bidirectional translator for both of these languages.
